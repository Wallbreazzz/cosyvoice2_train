# CosyVoice2 微调环境准备（需要外网）

本文档列出所有需要外网连接的下载安装命令。在有外网的环境中依次执行，完成后即可打包为离线镜像。

## 外网依赖总览

| 步骤 | 操作 | 外网目标 | 预估大小 |
|------|------|---------|---------|
| 1 | pip install 15个Python包 | PyPI | ~500MB |
| 2 | git clone CosyVoice + 子模块 | GitHub | ~100MB |
| 3 | 下载CosyVoice2-0.5B预训练模型 | ModelScope | ~3GB |
| 4 | 下载xunyi训练recipe（3个文件） | HuggingFace | ~10KB |

---

## 1. 安装Python依赖

```bash
pip install transformers==4.40.1 peft==0.11.1 \
  hyperpyyaml conformer==0.3.2 diffusers==0.29.0 \
  openai-whisper==20231117 pyworld==0.3.4 \
  soundfile==0.12.1 librosa==0.10.2 modelscope==1.15.0 \
  onnxruntime==1.18.0 pandas pyarrow omegaconf==2.3.0 \
  hydra-core==1.3.2 tensorboard==2.14.0 deepspeed==0.14.2 \
  torchvision==0.19.0 huggingface_hub==0.23.5
```

验证：
```bash
python3 -c "import torch; print('torch', torch.__version__)"
python3 -c "from peft import LoraConfig; print('peft OK')"
python3 -c "from huggingface_hub import cached_download; print('huggingface_hub OK')"
```

版本兼容性说明：
- `peft==0.11.1` 兼容 `transformers==4.40.1`（新版peft需要transformers>=4.43）
- `torchvision==0.19.0` 兼容 `torch==2.4.0`（Matcha-TTS安装时会拉错版本）
- `huggingface_hub==0.23.5` 兼容 `diffusers==0.29.0`（新版移除了cached_download）

## 2. 克隆CosyVoice代码

```bash
cd /home/mind/model/cosyvoice_train

git clone https://github.com/FunAudioLLM/CosyVoice.git

git fetch --depth 1 origin fd45708e4beb6ae40d1344452d7010cc338b0768

```

## 3. 下载预训练模型

```bash
cd /home/mind/model/cosyvoice_train/CosyVoice

# 修复 modelscope 在 Python 3.11+ 下的兼容性问题
sed -i 's/attr = getattr(node, field)$/attr = getattr(node, field, None)/' \
    /usr/local/lib/python3.11/site-packages/modelscope/utils/ast_utils.py

mkdir -p pretrained_models/CosyVoice2-0.5B

python3 -c "
from modelscope import snapshot_download
snapshot_download('iic/CosyVoice2-0.5B',
  local_dir='pretrained_models/CosyVoice2-0.5B')
"
```

验证：
```bash
ls pretrained_models/CosyVoice2-0.5B/llm.pt
ls pretrained_models/CosyVoice2-0.5B/flow.pt
ls pretrained_models/CosyVoice2-0.5B/hift.pt
ls pretrained_models/CosyVoice2-0.5B/campplus.onnx
ls pretrained_models/CosyVoice2-0.5B/speech_tokenizer_v2.onnx
ls pretrained_models/CosyVoice2-0.5B/CosyVoice-BlankEN/
```

## 4. 下载xunyi训练recipe

```bash
cd /home/mind/model/cosyvoice_train/CosyVoice

mkdir -p examples/libritts/cosyvoice2/conf

wget -q -O examples/libritts/cosyvoice2/conf/cosyvoice2.yaml \
    https://huggingface.co/xunyi/SMIIP-NV_finetune_CosyVoice2/resolve/main/examples/nv/cosyvoice2/conf/cosyvoice2.yaml

wget -q -O examples/libritts/cosyvoice2/run.sh \
    https://huggingface.co/xunyi/SMIIP-NV_finetune_CosyVoice2/resolve/main/examples/nv/cosyvoice2/run.sh

wget -q -O examples/libritts/cosyvoice2/conf/ds_stage2.json \
    https://huggingface.co/xunyi/SMIIP-NV_finetune_CosyVoice2/resolve/main/examples/nv/cosyvoice2/conf/ds_stage2.json
```
